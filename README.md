#StockGraph
A total implementation of stock graph,written by JavaScript and HTML5.


一个完整的股票K线图实现，使用JavaScript和Html5编写。
###K线类别：
- 1分钟
- 5分钟
- 15分钟
- 30分钟
- 60分钟
- 日K
- 周K
- 月K
- 年K
###分时图类别：
- 分时
- 五日
###市场范围：
- 中国大陆所有交易市场（包括A股、期货、中小板、文交所等）
- 港股所有交易市场
- 美股主要交易市场
###功能范围
- K线图、分时图
- 交易量图
- MA均线
- MACD指标
- 十字光标
- 左右滑、缩放
